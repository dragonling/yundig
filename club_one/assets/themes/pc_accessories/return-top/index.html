jquery.returnTop返回顶部Jquery插件

作者：似懂非懂
Blog：http://www.haw86.com

调用方法

<script type="text/javascript">
    $(function(){
    //调用返回顶部
    $("#returnTop").returnTop({right:40,bottom:30});
})
</script>
